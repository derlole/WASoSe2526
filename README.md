## Results
Im ordner `results` sind unterschiedliche files von den Rohdaten in `results_lang.csv` über Händische analysen in `AUSWERTUNG.md` und automatischn auswertungsscripts und deren resultat in `analyze.py` und `result.txt`.

## Code
Unter dem Ordner `Code` lassen sich sämtliche genberierten Code abschnitte durch ordner und Dateibenennung zuordnen, so entsprechen die 6 Auftrtetenden Ordner den 6 getesteten Modellen.
Die 15 Ordner, welche jeweils unter den Modell-ordnern zu finden sind der Aufgaben nummer, in welchen die ergebnisse der jeweiligen durchläufen zu finden sind.

Zum zweck der übersicht werden hier dem code hinzugefügte ungefragte erklährungen der Modelle nicht mit eingefügt, außer, das Modell erklährte explizit, warum eine gewisse Programmiersprache verwendet wurde.
